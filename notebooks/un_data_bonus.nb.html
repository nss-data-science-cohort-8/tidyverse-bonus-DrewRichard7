Introduction to the Tidyverse: UN Data - Bonus

Tidyverse Solo Exercise

In this project, you’ll practice working with data using the tidyverse libraries. You’ll be working with data on each of 145 school districts and the State of Tennessee. This data contains, for the 2014-2015 school year: * Proficiency rates on state tests * Student demographics * Chronic absenteeism * Discipline (suspension, expulsion) rates * High school graduation, dropout rates * Average ACT composite scores * A region in Tennessee
